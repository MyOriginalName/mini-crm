Mini CRM API Документация

Аутентификация

Регистрация пользователя

POST /api/v1/register

{
  "name": "Иван Иванов",
  "email": "ivan@example.com",
  "password": "securepassword",
  "password_confirmation": "securepassword"
}

Ответ:

{ "token": "your_access_token" }

Авторизация пользователя

POST /api/v1/login

{
  "email": "ivan@example.com",
  "password": "securepassword"
}

Ответ:

{ "token": "your_access_token" }

Выход из системы

POST /api/v1/logout

Ответ:

{ "message": "Successfully logged out" }

Получение информации о пользователе

GET /api/v1/user

Ответ:

{
  "id": 1,
  "name": "Иван Иванов",
  "email": "ivan@example.com"
}

Клиенты

Получение списка клиентов

GET /api/v1/clients

Ответ:

[{ "id": 1, "name": "Иван Иванов", "email": "ivan@example.com", "phone": "+79991234567" }]

Создание клиента

POST /api/v1/clients

{ "name": "Петр Петров", "email": "petr@example.com", "phone": "+79991112233" }

Обновление данных клиента

PUT/PATCH /api/v1/clients/{client}

{ "name": "Новый Имя", "email": "newemail@example.com" }

Удаление клиента

DELETE /api/v1/clients/{client}

Задачи

Получение списка задач

GET /api/v1/tasks

Создание новой задачи

POST /api/v1/tasks

{ "client_id": 1, "title": "Подготовить презентацию", "description": "Сделать презентацию для клиента" }

Получение конкретной задачи

GET /api/v1/tasks/{task}

Обновление задачи

PUT/PATCH /api/v1/tasks/{task}

{ "title": "Обновленный заголовок", "status": "completed" }

Удаление задачи

DELETE /api/v1/tasks/{task}
